#YARD REPORT — FEATURE INVENTORY
// Current site capabilities. Update when features ship.

#News Feed
  - RSS aggregation from 14 baseball sources (Orioles + MLB): Orioles, Baltimore Baseball, Camden Chat, Birds Watcher, Eutaw Street Report, MASN Sports, MLB.com, MLB Trade Rumors, ESPN MLB, FanGraphs, Baseball Prospectus, Baseball America, New York Times Baseball, and The Athletic
  - 7-day default article feed with selectable range (3 / 7 / 14 / 30 days)
  - Full article reader (iframe for full RSS content, excerpt + link for others)
  - Off-topic article filter (rejects non-baseball content from general feeds)
  - Filler image detection (replaces generic thumbnails with YR favicon)
  - Around the Horn story bundling (max 3, gradient glow, descriptive phrase titles with "Player Name:" prefix when detected, expandable)
  - Source favicon + text on each card
  - Share button on articles (native share on mobile, clipboard on desktop)
  - Thumbnails standardized to 16:9 aspect ratio
  - Read/unread tracking (green ✓ tick, dimmed cards, persisted to localStorage, cap 200)
  - Swipe left to mark read, swipe right to unmark (mobile touch)
  - Read articles auto-hide after 5 seconds
  - Default view: list

#Views & Filters
  - Category filter pills (All / Orioles / MLB / MiLB) left-justified in feed toolbar
  - Grid, list, and compact view modes
  - Source filter and view mode icons right-justified in feed toolbar
  - Mobile defaults to list view (grid hidden on small screens)
  - Sort by newest, by date group, or by source
  - Date range selector (3 / 7 / 14 / 30 days)
  - MiLB auto-detection (prospect, minor league, farm system articles tagged and filtered)
  - Source filter aligned with view controls (popover in toolbar, bottom-sheet on mobile)
  - Search across all articles

#Score Bugs
  - Yesterday / Today / Tomorrow games
  - Yesterday's scores stay prominent until 09:00 EDT
  - Orioles games highlighted with orange accent
  - Live game indicator with pulsing dot + inning
  - Rain Delay and Postponed states called out with dedicated chip styling (blue / red)
  - Base runner diamond during live games (correctly oriented 1st/2nd/3rd, amber bases when occupied)
  - Weather emoji + temperature inline for preview games (Open-Meteo forecasts)
  - Box score popover on hover (inning-by-inning linescore, R/H/E, W/L/SV, top 3 hitters & pitchers)
  - Links to MLB Gameday (preview or final)
  - Auto-refresh every 5 minutes
  - Pitcher scouting notes with pitch arsenal (pitch type, usage %, velocity) for preview games
  - Team logos inline with Key Hitters names (removed abbreviations)
  - Team logos in lineup popover titles with full team names (Baltimore Orioles)
  - Batter/pitcher handedness (L/R) indicators next to player names
  - Orioles player names in lineup and pitching rows include walk-up song icons
  - Live Orioles games can show an Orioles Walk-Up Queue for At Bat, On Deck, In Hole, or Warmup players
  - Pitchers sorted by innings pitched (starters first)
  - Scout Notes for all game states (preview/live/final)
  - Scout Notes show probable pitcher matchup for preview games
  - Scout Notes show final result summary for final games

#Walk-Up Songs
  - Dynamic source: official Orioles walk-up music page via proxy text fetch
  - 6-hour in-memory cache for live song metadata
  - Parser reads Next.js __NEXT_DATA__ first, then falls back to player-link + Spotify iframe/anchor DOM order
  - Complete April 2026 fallback map covers 23 active Orioles plus 6 IL/inactive players
  - Multiple songs per player supported (ex: Gunnar Henderson)
  - Spotify-style icons render next to Orioles player names in Roster, score lineup rows, pitching rows, and live walk-up queue
  - Known song icons open an embedded Spotify player overlay with an "Open in Spotify" link
  - Missing song icons open a Spotify search overlay so every Orioles roster player still has an action
  - Roster renders immediately with fallback/cache data, then re-renders when live song fetch completes

#Left Sidebar
  - Search
  - Podcast: compact audio player for the latest Baseball Tonight with Buster Olney episode, placed above Video with episode/detail/archive links
    - RSS feed: https://feeds.megaphone.fm/ESP1723897648 (fetched directly in browser, CORS open)
    - Archive/details page: https://www.espn.com/espnradio/podcast/archive?id=2386164
    - Hover to see episode title and description
  - Video: latest video from MLB Fastcast, MLB Top Plays, Orioles Game Recaps, and a random Orioles Moments video on each page load, play icon overlay on thumbnails, theater mode overlay with YouTube embed on click
    - YouTube playlist feeds: MLB Fastcast, MLB Top Plays, Orioles Game Recaps, Orioles Moments
    - Orioles Game Recaps fall back to MLB game media for the latest completed Orioles game when the Orioles YouTube playlist is stale
  - On Deck: next Orioles game with opponent, date/time, venue, weather forecast, lineup popover on hover (game day)
  - Schedule: horizontal row of game boxes (day, opponent logo, @ indicator for away), links to MLB preview (merged into On Deck widget)
  - Standings: all 6 divisions with division tabs, Orioles highlighted, team abbreviations link to MLB.com

#Right Sidebar
  - Yard Leaders: O's / AL / NL / MLB / ROY scope pills, batting and pitching groups, player names link to Baseball Savant, stat labels link to corresponding Baseball Savant leaderboards/custom stat views
    - Batting: AVG, OBP, OPS, HR, H, BB, SLG, RBI
    - Pitching: ERA, SO, QS, WHIP, W, K/9, BB/9
  - Roster: 40-man Orioles roster grouped by position players, pitchers, injured list, and minors; jersey numbers, links to Baseball Savant, walk-up song/search icons (default collapsed)
  - Injury Report: current IL players with position, IL type (10/15/60-day), and injury note; player names link to MLB.com
  - Transactions: last 14 days of Orioles moves, sorted newest-first, player names link to MLB.com, auto-refreshes every 5 minutes (positioned below Injury Report)
  - Contracts: curated FanGraphs Roster Resource and Spotrac payroll links

#Settings
  - Settings gear icon in header opens overlay panel
  - Theme: Light / Dark / System / City Connect (persisted to localStorage, system follows OS)
  - Default view: Grid / List / Compact (persisted to localStorage)

#Design
  - Dark theme (default), light theme available, City Connect theme selectable and auto-applied on Fridays
  - 3-column layout (left sidebar / feed / right sidebar)
  - Widget titles prominent (0.78rem, white, uppercase)
  - All sidebar sections have inline icons and are collapsible
  - Sticky header with Yard Report logo
  - Responsive: 3-col > 2-col at 1100px > 1-col at 600px (mobile)
  - Mobile: compact header, right sidebar below feed, sidebar sections auto-collapsed, safe area support
  - Cross-browser optimized (Chrome, Safari, Firefox), iOS momentum scroll, webkit prefixes
  - Both left and right sidebars independently scrollable with sticky positioning
  - ATH cards scale down on mobile (2-col at 600px, 1-col at 400px)

#Infrastructure
  - GitHub Actions CI/CD is the official deploy path (build + lftp deploy to `public_html/yardreport`)
  - cPanel Git Version Control remains available only as a manual fallback path
  - Cron guidance: keep scheduled tasks minimal; prefer deploy-time smoke tests and daily health/proxy checks over scheduled content refresh jobs
  - Scheduled GitHub Actions checks now run daily for smoke + homepage health and for production proxy/feed sanity
  - PHP RSS proxy with domain allowlist (includes youtube.com and megaphone.fm), audio-enclosure extraction, and optional raw text/HTML passthrough for trusted pages
  - Official Orioles walk-up music page for dynamic walk-up song metadata
  - Spotify track/search embeds for walk-up song player overlay
  - MLB Stats API for scores, standings, schedule, roster, transactions, leaders, boxscore
    - Pitcher pitch arsenal: `/people/{playerId}/stats?stats=pitchArsenal&season=YYYY&group=pitching`
  - Open-Meteo API for weather forecasts (all 30 MLB venues)
  - Google Favicon API for source icons
  - YouTube RSS feeds for video widget
  - Megaphone RSS feed for podcast widget (fetched directly in browser)
  - ESPN podcast archive page for podcast detail links
  - MLB.com Gameday, roster, player, and transaction pages for outbound links
  - MLBstatic team-logo assets for team badges and On Deck visuals
  - Baseball Savant player and leaderboard pages for Yard Leaders and roster links
