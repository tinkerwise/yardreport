Yard Report Changelog
Week of March 28 - April 3, 2026

April 3, 2026

Masthead
- Reworked the Yard Report masthead to sit more cleanly inline with the full-width header instead of crowding the score bar.
- Kept the logo visually larger while slimming its horizontal footprint.
- Replaced the previous Yard wordmark font with a cleaner athletic display face and refined the Camden-inspired banner treatment.

Lineups
- Added lineup-card content to the score bug hover popover so each game chip can surface posted/not-posted lineup state.
- Updated the On Deck lineup indicator to always show the lineup-card icon, grey when unavailable and green when at least one lineup is posted.
- Tightened lineup popover rows so player names stay on one line with smaller type and truncation instead of wrapping.
- Expanded the lineup popovers so names fit more comfortably and gave them a warmer distinct background treatment.
- Simplified the On Deck lineup badge to an icon-only state without the numeric `2/2` counter.
- Added probable pitchers to lineup popovers so the starters appear even before lineups are posted.
- Reworked the On Deck lineup popover so it can expand over the page instead of being constrained to the sidebar width.
- Standardized the lineup/box popover surfaces to use stronger contrasting backgrounds against the rest of the page.

Light Theme
- Improved light-mode masthead contrast so the header wordmark and banner treatment stand apart more clearly from the page background.

Right Sidebar
- Let the right-side widgets expand to their full content height instead of clipping to the viewport.
- Removed the remaining internal height caps from the Roster and Transactions lists so both widgets expand to full content on desktop and mobile.

Yard Leaders
- Stacked batting and pitching leaders into one expanded widget and removed the separate batting/pitching toggle labels.

Feed and Toolbar
- Tightened the main feed toolbar layout.
- Left-justified the category pills (All / Orioles / MLB / MiLB).
- Right-justified the source filter and view controls.
- Removed the article count and "Last X days" helper text above the feed for a cleaner top-of-feed presentation.

Score Bugs
- Added dedicated score-chip states for Rain Delay and Postponed games.
- Delay/postponement states now get their own labels and visual accents instead of falling back to generic preview/live/final handling.

Yard Leaders
- Updated stat-label links so batting stats open the corresponding Baseball Savant leaderboard/custom stat view.
- Reworked pitching stat links against the live Baseball Savant site.
- Exact Savant custom-column links now support ERA, SO, W, QS, and GS.
- WHIP, K/9, and BB/9 now route to the closest valid Savant custom views built from the live available pitcher fields.
- Changed the pitching label from K to SO.

Right Sidebar
- Reordered the right sidebar so Injury Report now appears above Transactions.
- Expanded the Injury Report widget to include injury notes in a more compact layout.
- Injury notes from the roster feed are now displayed under each injured player when available.

Video Widget
- Removed Jomboy Breakdowns from the video rotation.
- Added Orioles Game Recaps.
- Added The Chill.
- Added Orioles Moments as a random-per-load playlist entry so a different Orioles highlight can appear on each page refresh.

Easter Eggs
- Improved the "magic" Easter egg:
  - the bird now appears centered in the viewport
  - confetti repeats for the full visible/audio duration instead of firing once
- Added a new "heritage" Easter egg trigger.
- Typing "heritage" in search now launches a temporary Heritage Edition mode:
  - forces grid view during the effect
  - transforms feed cards into vintage baseball-card-inspired designs
  - shows a Heritage Edition badge
  - restores the user's previous view after the effect ends

Infrastructure
- Extended the RSS proxy to optionally return raw text/HTML for trusted pages, not just feed XML.
- Added dev/prod support for using that proxy path to enrich client widgets with trusted page content.

Documentation
- Updated `README.md`, `brief.txt`, `updates.txt`, and `changelog.txt` to reflect the current shipped behavior, including toolbar alignment changes, score-bug delay states, Yard Leaders updates, the enhanced IL widget, the revised video widget, and the new Heritage Easter egg.
