#Yard Report
The Yard Report is a personal baseball news aggregator site primarily covering the Orioles and the rest of Major League Baseball. 

###Feature Set
- Articles from trusted sources
- On Deck & Upcoming Schedule widgets
- Division standings widget
- Roster widget (links to BaseballSavant)
- Transactions widget
- IL widget

##Updates
Updates to site are added to the FEATURES.md file, when prompted Claude reads, develops and executes a plan to implement the updates on site.
