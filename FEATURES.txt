#YARD REPORT

##Features & Ideas

**Rules of engagement:**
- Don't break the site
- Prioritize visual design and readability
- be aware of content overload, always suggest condensing content in some way
- anticipate visual cues for visitors to types of media content (text, audio, video, graphics)
- minimze load on browser cache, memory, etc.


#News Feed
  - RSS aggregation from 14 baseball sources (Orioles + MLB)
  - 3-day default article feed with selectable range (3 / 7 / 14 / 30 days)
  - Full article reader (iframe for full RSS content, excerpt + link for others)
  - Off-topic article filter (rejects non-baseball content from general feeds)
  - Filler image detection (replaces generic thumbnails with YR favicon)
  - Around the Horn story bundling (max 3, gradient glow, descriptive phrase titles with "Player Name:" prefix when detected, expandable)
  - Source favicon + text on each card
  - Share button on articles (native share on mobile, clipboard on desktop)
  - Thumbnails standardized to 16:9 aspect ratio
  - Read/unread tracking (green ✓ tick, dimmed cards, persisted to localStorage, cap 200)
  - Swipe left to mark read, swipe right to unmark (mobile touch)
  - Read articles auto-hide after 5 seconds
  - Default view: list

#Views & Filters
  - Category filter pills (All / Orioles / MLB / MiLB) left-aligned in feed toolbar
  - Grid, list, and compact view modes
  - View mode icons right-aligned in feed toolbar
  - Mobile defaults to list view (grid hidden on small screens)
  - Sort by newest, by date group, or by source
  - Date range selector (3 / 7 / 14 / 30 days)
  - MiLB auto-detection (prospect, minor league, farm system articles tagged and filtered)
  - Source filter right-aligned with view controls (popover in toolbar, bottom-sheet on mobile)
  - Search across all articles

#Score Bugs
  - Yesterday / Today / Tomorrow games
  - Yesterday's scores stay prominent until noon EDT
  - Orioles games highlighted with orange accent
  - Live game indicator with pulsing dot + inning
  - Rain Delay and Postponed states called out with dedicated chip styling
  - Base runner diamond during live games (correctly oriented 1st/2nd/3rd, amber bases when occupied)
  - Weather emoji + temperature inline for preview games (Open-Meteo forecasts)
  - "Story" link on final scores linking to MLB game story
  - Box score popover on hover (inning-by-inning linescore, R/H/E, W/L/SV, top 3 hitters & pitchers)
  - Links to MLB Gameday (preview or final)
  - Auto-refresh every 5 minutes

#Left Sidebar
  - Search
  - Video: latest video from MLB Fastcast, MLB Top Plays, Jomboy Breakdowns (YouTube RSS), play icon overlay on thumbnails, theater mode overlay with YouTube embed on click
  - On Deck: next Orioles game with opponent, date/time, venue, weather forecast, lineup popover on hover (game day)
  - Schedule: horizontal row of game boxes (day, opponent logo, @ indicator for away), links to MLB preview (merged into On Deck widget)
  - Standings: all 6 divisions with division tabs, Orioles highlighted, team abbreviations link to MLB.com

#Right Sidebar
  - Yard Leaders: batting/pitching toggle, player names link to Baseball Savant, stat labels link to corresponding Baseball Savant leaderboards/custom stat views
    - Batting: AVG, OBP, OPS, HR, H, BB, SLG, RBI
    - Pitching: ERA, SO, GS, QS, WHIP, W, K/9, BB/9
  - Roster: active Orioles roster grouped by position, jersey numbers, links to Baseball Savant (default collapsed)
  - Injury Report: current IL players with position, IL type (10/15/60-day), projected return date, and injury note; player names link to MLB.com
  - Transactions: last 14 days of Orioles moves, sorted newest-first, player names link to MLB.com, auto-refreshes every 5 minutes (positioned below Injury Report)

#Settings
  - Settings gear icon in header opens overlay panel
  - Theme: Light / Dark / System (persisted to localStorage, system follows OS)
  - Default view: Grid / List / Compact (persisted to localStorage)

#Design
  - Dark theme (default), light theme available
  - 3-column layout (left sidebar / feed / right sidebar)
  - Widget titles prominent (0.78rem, white, uppercase)
  - All sidebar sections have inline icons and are collapsible
  - Sticky header with Yard Report logo
  - Responsive: 3-col > 2-col at 1100px > 1-col at 600px (mobile)
  - Mobile: compact header, right sidebar below feed, sidebar sections auto-collapsed, safe area support
  - Cross-browser optimized (Chrome, Safari, Firefox), iOS momentum scroll, webkit prefixes
  - Scrollable sidebars

#Easter Eggs
  - Type "magic" + Enter in search: random Oriole bird graphic centered in viewport with repeating confetti animation, Orioles Magic audio clip, display lasts for audio duration
  - Type "heritage" + Enter in search: temporary Heritage Edition mode with baseball-card-themed feed cards, force grid view during effect, then restore prior view
  - Konami Code (↑↑↓↓←→←→BA): Seven Nation Army chant with logo animation
  - Triple-click theme toggle: secret OPACY theme (all-orange dark mode)

#Infrastructure
  - Astro static site deployed to cPanel via Git Version Control
  - GitHub Actions CI/CD (build + lftp deploy)
  - PHP RSS proxy with domain allowlist (includes youtube.com) and optional raw text/HTML passthrough for trusted pages
  - MLB Stats API for scores, standings, schedule, roster, transactions, leaders, boxscore
  - Open-Meteo API for weather forecasts (all 30 MLB venues)
  - Google Favicon API for source icons
  - YouTube RSS feeds for video widget
